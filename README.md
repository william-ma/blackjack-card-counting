## Blackjack Card Counting

A web app used to train and teach card counting in blackjack.

Developed using:
- [x] Java Spring Boot (Web framework)
- [x] MongoDB (NoSQL database)
- [x] Gradle (Dependency management)

### Card Counting Strategies

- [x] Hi-Lo
- [x] Hi-Opt I
- [x] Hi-Opt II
- [x] KO
- [x] Omega II
- [x] Red 7
- [x] Halves
- [x] Zen Count
